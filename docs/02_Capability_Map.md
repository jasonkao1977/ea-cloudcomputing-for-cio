# 02. Capability Map

## 視圖概述
定義企業為達成戰略目標所需具備的核心能力。每個能力都標註了 KPI 與成熟度，作為投資與改善的依據。

| 能力名稱 | 擁有者 | KPI 指標 |

| **資料平台** | 資料長 (CDO) | 1. 資料品質分數 > 90%<br>2. [cite_start]資料攝取延遲 < 15 分鐘 [cite: 5] |
