# 08. Service Map

## 視圖概述
企業內部的 IT 服務目錄。將所有可用服務依照領域 (Domain) 進行分類，促進服務重複使用。

## 服務領域 (Domains)
* [cite_start]**Business Domain**: 企業雲服務目錄, 資料平台即服務 [cite: 8-9]。
* **Platform Domain**:
    * [cite_start]應用服務: 身份驗證, 金鑰管理, 事件匯流 [cite: 12-13]。
    * [cite_start]技術服務: 容器與Kubernetes, 可觀測性 [cite: 20-22]。
* [cite_start]**Data Domain**: 資料查詢, 儲存, 資料庫 [cite: 14, 20]。
* [cite_start]**AI Services**: Prompt Management, AI Gateway, RAG Engine [cite: 117]。
